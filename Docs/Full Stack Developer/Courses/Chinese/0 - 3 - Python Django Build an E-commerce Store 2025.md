欢迎！我来这里是为了帮助你用 Django 建立自己的电子商务商店。

——请仔细阅读——

本课程主要面向那些对 Django 有基本了解的人。你还需要了解一些 HTML、CSS、JavaScript 知识，一点 bootstrap 知识会很有帮助。如果你满足这些条件，那么本课程就适合你！

本课程开发的电子商务商店项目仅用于学习目的。我将教你如何建立一个具有实时商店预期自定义功能的电子商务商店。
