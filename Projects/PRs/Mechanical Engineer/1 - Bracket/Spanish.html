Análisis estático lineal de un soporte

Se realizó un análisis estático lineal de un soporte utilizando Abaqus. El modelo contiene 2.849 elementos y 4.144 nodos, con las propiedades del material definidas para el aluminio. Se aplicaron condiciones de contorno y restricciones de acoplamiento en varios puntos del soporte, incluidos puntos de referencia para orificios de orejetas. El análisis utilizó elementos C3D8R y tuvo como objetivo estudiar la respuesta estructural bajo carga estática. Los resultados mostraron que el modelo se ejecutó correctamente.
